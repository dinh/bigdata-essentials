### Moving Cursor

| Description      | Command       |
| ------------- |:-------------:| 
| Move to the end of the file      | :$ (or) G | 
| Jump to beginning of the file     | gg (or) 1G     |  
| Jump to line 50 | 50G     |   


### File modifications

| Description      | Command       |
| ------------- |:-------------:| 
| Delete lines not starting with Warning      | v/^Warning/d | 
| Open multiple files at once in tabs    | vim -p file1 file2 (gt to to next tab)     |
